## 1. 到「設定」 > 「捷徑」將「允許不受信任的捷徑」點選開啟
註1：iOS 本身有限制必須先執行過另一個捷徑才能開啟本設定<br/>
註2：加完本捷徑後可自行關閉<br/>
<p align="left"><img width="400" src="./2.png"></p><br/>
